# Test PR Tracker Repository

Repository này được tạo ra để thử nghiệm công cụ **GitHub PR Tracker** của bạn.

## Hướng dẫn test PR:
1. Tạo một nhánh mới (ví dụ: `feature/test-webhook`).
2. Sửa file này hoặc thêm file mới.
3. Commit và Push lên remote.
4. Tạo Pull Request (PR) từ nhánh `feature/test-webhook` vào nhánh `main`.
5. Quan sát thông báo nhảy về kênh Discord của bạn!
